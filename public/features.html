Features

機能一覧

40以上のサービスモジュールで構成されるフルスタックP2Pプラットフォーム
🔄

TruPay自動ポーリング

1分間隔でTruPay APIから承認済み出金を自動検知。Merchant(pay@sloten.io)フィルタ + 国内銀行振込のみ抽出。

CORE
🤝

P2P自動マッチング

30秒間隔で出金キューとUSDT購入者を金額レンジでマッチング。CAS(Compare-and-Swap)で二重マッチ防止。Mutex付き。

CORE
🤖

AI振込明細解析

Claude Vision APIで振込スクショをOCR。銀行名・口座番号・名義・金額を抽出しスコアリング(0-100)。手数料込み金額も許容。

AI
📊

リアルタイムレート比較

Bybit・Binance・OKXのP2Pレートを30秒更新。USDT/BTC/ETH対応。スプレッド・乖離率・アービトラージ機会を自動検出。

MARKET
🔐

JWT自動リフレッシュ

TruPay APIトークンを12時間ごとに自動更新。401検知時は即座にリログイン+リトライ。Mutex付きで並行ログイン防止。

CORE
✅

着金自動確認

TruPay status=32(completed)を毎分ポーリングで検知。手動確認(scrapper/match API)も対応。購入者報告→AI解析→自動/手動承認。

CORE
💎

USDT自動送金

TRC-20 USDT送金。3回リトライ+指数バックオフ。CASで二重送金防止。Circuit Breakerでブロックチェーン障害保護。

CORE
🔔

Telegram通知

全イベントをリアルタイム通知。マッチング成立・AI解析結果(画像付き)・着金確認・USDT送金・タイムアウト。承認/却下ボタン付き。

NOTIFY
🤖

Telegram購入Bot

Bot内で完結する購入フロー。金額選択→ウォレット入力→マッチング→振込先通知→完了報告。取引所レート購入モードも搭載。

CHANNEL
🌐

Web UI購入ページ

アカウント不要のUSDT購入ページ。信頼バッジ・ステップ表示・タイマー・振込先コピー・スクショ必須アップロード・状態保存(localStorage)。

CHANNEL
📢

マーケティング自動配信

Telegram公開チャンネル+X/Twitterに2時間ごと自動投稿。4言語ローテーション(日/英/中/越)。レート・マッチング件数・CTA付き。

GROWTH
📈

マッチング候補ダッシュボード

管理画面でTruPay出金キュー × P2P買い注文の金額レンジマッチング候補を自動分析。需要/供給比率・推奨レート表示。

GROWTH
💱

USDT売却者(セラー)システム

セラーがUSDTをデポジット→購入者に自動供給。自社ウォレット不要の完全P2Pモデル。取引数順の自動セラー選択。

P2P
🛡️

CSP Nonce セキュリティ

リクエストごとにランダムnonce生成。unsafe-inline完全除去。Cloudflare + Caddy + Express 3層セキュリティ。

SECURITY
🗄️

DB永続化 + 自動バックアップ

SQLite WALモード + busy_timeout。購入者キューDB永続化(再起動復元)。Dockerサイドカーで日次自動バックアップ(7日保持)。

Architecture

技術アーキテクチャ

Node.js + TypeScript モノリスで高速開発。Docker単一コンテナ + サイドカーバックアップ。

Backend

  • Node.js 20 + TypeScript + Express
  • SQLite (better-sqlite3) WAL mode
  • TronWeb (TRC-20 USDT送金)
  • Multer (ファイルアップロード)
  • WebSocket (リアルタイム更新)

外部API連携

  • TruPay API (出金検知・着金確認)
  • Bybit / Binance / OKX P2P API
  • TronGrid API (ブロックチェーン)
  • Claude Vision API (AI OCR)
  • Telegram Bot API
  • X/Twitter API v2

セキュリティ

  • CSP Nonce (per-request)
  • CSRF Double-Submit Cookie
  • bcrypt パスワードハッシュ
  • AES-256 データ暗号化
  • SQL列名ホワイトリスト
  • CAS二重送金防止(3層)

インフラ

  • Docker + Caddy + Cloudflare
  • Hetzner VPS (シンガポール)
  • メモリ制限 2GB / CPU 2コア
  • 日次SQLiteバックアップ(7日保持)
  • Prometheus メトリクス
  • 構造化ログ (JSON Lines)
