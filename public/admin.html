API 使用方法

認証: 全リクエストに X-API-Key: <your_key> ヘッダーを付与

注文作成:

POST /api/v1/orders
X-API-Key: bkpay_sk_live_...

{
  "amount": 10000,
  "crypto": "USDT",
  "customerWalletAddress": "T..."
}

レスポンス例:

{
  "success": true,
  "orderId": "ORD-XXXX-YYYY",
  "bankAccount": {
    "bankName": "SBI銀行",
    "accountNumber": "1234567",
    "accountHolder": "BK STOCK",
    "amount": 10000
  },
  "cryptoAmount": 12.34,
  "rate": 810.5,
  "expiresAt": 1234567890000
}

ステータス確認: GET /api/v1/orders/:orderId

残高確認: GET /api/v1/balance

Webhook ペイロード（注文完了時）:

{
  "event": "order.completed",
  "orderId": "ORD-XXXX-YYYY",
  "status": "completed",
  "amount": 10000,
  "crypto": "USDT",
  "cryptoAmount": 12.34,
  "txId": "abc123...",
  "timestamp": 1234567890000
}
// Header: X-BKPay-Signature: sha256=HMAC(webhookSecret, body)

セラー登録 URL
セラーは以下のエンドポイントから自己登録できます: 
POST /api/p2p/sellers/register
{
  "name": "田中太郎",
  "email": "seller@example.com",
  "password": "securepass123",
  "paypayId": "tanaka-t-1234",
  "minAmount": 1000,
  "maxAmount": 100000,
  "payMethods": ["paypay"]
}

登録後は管理者が承認（ステータス: active）してから取引可能になります。

セラー確認ページ: /seller-confirm.html?orderId=ORD-xxx&token=SELLER_TOKEN
